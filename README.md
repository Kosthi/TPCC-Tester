# TPCC-Tester

python3 -m tpcc.main --scale 1 --init --benchmark --threads 16 --transactions 300

本项目为 2023-2024 全国大学生计算机系统能力大赛-数据库管理系统设计赛决赛模拟评测脚本，由 DataDance 团队维护和开发。

欢迎未来的数据库大赛参赛选手使用和PR。
